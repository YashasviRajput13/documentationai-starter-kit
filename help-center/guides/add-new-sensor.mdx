## Connect a Sensor

<Steps>
  <Step title="Go to Sensors Page">
    Log in, then select **Sensors** from the side menu.
  </Step>
  <Step title="Click 'Add Sensor'">
    Use the **Add Sensor** button at the top right.
  </Step>
  <Step title="Enter Sensor Details">
    Fill in: Name, Type, Location. Click **Save**.
  </Step>
  <Step title="Verify Connection">
    Sensor status should show as **Online** within a few minutes.
  </Step>
</Steps>
