## Upgrading Neuro Grid

Follow these steps to upgrade your Neuro Grid instance.

<Steps>
  <Step title="Backup Data" icon="database">
    Export your configuration and telemetry data.

````bash
neurogrid-cli export --output=backup.json
````
  </Step>
  <Step title="Update CLI" icon="download">
    Install the latest CLI version.

<CodeGroup tabs="npm,yarn">
````bash
npm install -g @neurogrid/cli@latest
````
````bash
yarn global add @neurogrid/cli@latest
````
</CodeGroup>
  </Step>
  <Step title="Run Upgrade" icon="rocket">
    Apply the update.

````bash
neurogrid-cli upgrade --version=1.3.0 --api-key=YOUR_API_KEY
````
  </Step>
</Steps>
